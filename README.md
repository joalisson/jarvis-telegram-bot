# jarvis-python-telegram-bot

Extremely intelligent bot, able to talk and interact with you without thinking twice

### How do Run
Install virtualenv
```
$ virtualenv venv
```
Activate virtualenv
```
$ . venv/bin/activate
```
Install dependecies
```
$ pip install -r requirements.txt
```
Run python script
```
$ python jarvisbot.py
```
